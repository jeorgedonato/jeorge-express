### Objective 
To make a responsive web portfolio that uses MVC framework

### Technologies Used:

- **HTML5**
- **CSS3**
- **Javascript**
- **Nodejs**
- **Express**
- **Bulma CSS Framework**
- **Express-Handlebars**

### Installation

``` sh
"npm install" to install node dependecies
```

### Usage

``` sh
"npm start" to run the script.
```
